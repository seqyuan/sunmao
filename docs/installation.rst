Installation
============

Installing Sunmao
-----------------

Sunmao can be installed using pip:

.. code-block:: bash

    pip install sunmao

Or using Poetry:

.. code-block:: bash

    poetry add sunmao

Requirements
------------

Sunmao requires:

- Python 3.8 or higher
- matplotlib >= 3.5.0
- numpy >= 1.21.0

Development Installation
------------------------

To install Sunmao in development mode:

.. code-block:: bash

    git clone https://github.com/seqyuan/sunmao.git
    cd sunmao
    pip install -e .

Or with Poetry:

.. code-block:: bash

    git clone https://github.com/seqyuan/sunmao.git
    cd sunmao
    poetry install
